Disabling Detectors

You can disable specific detectors using the stacksage.yml configuration:

exclusions:
  detectors:
    - ec2_generation_upgrade  # Disable this detector
    - gp2_to_gp3_migration    # And this one

Detector Confidence Levels

Each finding includes a confidence score (0.0-1.0) indicating accuracy:

  • 0.90-1.00: Very high confidence (e.g., unattached EBS volumes)
  • 0.75-0.89: High confidence (e.g., low CloudWatch metrics)
  • 0.60-0.74: Medium confidence (requires validation)
